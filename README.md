#LI KEVIN  
#My Personal Project : Top-Down Shooter   
  
Les touches de déplacement sont Z,Q,S,D. On peut tirer au coup par coup avec le clic gauche, ou maintenir pour tirer en continu. La touche R sert au rechargement. En bas à droite se situe le nombre de balles restantes dans le chargeur. La difficulté (vitesse des zombies + fréquence de spawn) augmente tous les 10 zombies tués.  
  
Il y a des warnings dans la console mais après avoir cherché, ils sont liés à l'utilisation du [SerializeField] mais rien de grave apparemment.  
  
L'exécutable se trouve dans le dossier "Build".
